# WOLOF - REFERENCE LINGUISTIQUE
# ================================
# Vocabulaire agricole Wolof pour les reponses de l'agent.
# Le Wolof est la langue la plus parlee au Senegal (~80% population).
# Ecriture en alphabet latin (pas arabe).

## Salutations
- Nanga def? = Comment vas-tu?
- Mangi fi rekk = Je suis la (je vais bien)
- Jere jef = Merci
- Ba beneen = Au revoir

## Agriculture
- Tool = Champ / culture
- Nguur = Terre / sol
- Taw = Pluie
- Noor = Soleil
- Ndox = Eau
- Gerte = Arachide
- Dugub = Mil
- Malo = Riz
- Mbaxal = Mais
- Niebe = Niebe (meme mot)
- Tamaate = Tomate
- Soble = Oignon

## Meteo
- Taw bi = La pluie
- Taw dina wax = Il va pleuvoir
- Taw baxul = Pas de pluie
- Noor bi tang na = Le soleil est chaud
- Ngelaw = Vent
- Feecc = Saison seche
- Nawet = Hivernage (saison des pluies)

## Actions agricoles
- Bey = Cultiver / labourer
- Roose = Arroser / irriguer
- Faj = Traiter / soigner
- Rang = Recolter
- Jaay = Vendre
- Jend = Acheter

## Maladies / Problemes
- Jegge = Maladie (plante)
- Gunaar = Insecte / ravageur
- Souf = Secheresse
- Ndox bu bari = Inondation (trop d'eau)

## Periodes
- Nawet = Saison des pluies (juin-octobre)
- Noor = Saison seche chaude (mars-juin)
- Lolli = Saison seche froide (novembre-fevrier)

## Phrases utiles pour l'agent
- "Sa tool yi dina am solo" = Tes cultures vont bien se porter
- "Waxal sa gerte yi ci kaw" = Prends soin de tes arachides
- "Taw dina am ci __ fan" = Il y aura de la pluie dans __ jours
- "Nangula sa tool yi" = Arrose tes cultures
- "Fajal sa tool yi" = Traite tes cultures
- "Jegeelu ci nawet bi" = Prepare-toi pour l'hivernage
- "Njeg bu baax la ngir jaay" = C'est un bon moment pour vendre

## Commandes SMS prevues
- METEO [ville] = Previsions meteo
- JEGGE [culture] = Diagnostic maladie
- NJEG [culture] = Prix du marche
- NDIMBAL = Aide / commandes disponibles
- TOOL [culture] = Conseils de culture

## Format reponse SMS (max 160 caracteres)
Les reponses en Wolof doivent etre courtes et directes.
Exemple : "Taw baxul ci 5 fan yi. Nangula sa gerte yi. Tang dina am 35C."
(Pas de pluie 5 jours. Arrose tes arachides. Il fera 35C.)
